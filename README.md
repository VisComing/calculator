# 高级科学计算器 (Advanced Scientific Calculator)

一个基于 Next.js 的全功能科学计算器，支持普通计算、程序员计算、数学函数计算以及积分求导。

## 功能特性

- **标准计算器**: 基础四则运算、百分比、括号运算
- **程序员计算器**: 进制转换、位运算、逻辑运算
- **科学计算器**: 三角函数、对数、指数、阶乘、排列组合等
- **微积分计算**: 符号积分、求导、极限计算
- **美观界面**: 现代化 UI 设计，支持深色/浅色主题

## 技术栈

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- mathjs (数学计算)
- Algebrite (符号计算 - 积分求导)
- shadcn/ui (UI 组件)
